Designers And Developers

No Longer A House Divided

Any modern web design process requires intense collaboration between designers and front-end developers

This collaboration and empathy between disciplines are just as important as the techniques, methods and development frameworks used to take apart a problem.

We need to know each other tools

Developers should be able to understand a layout, having a detailed knowledge of it and designers should have an idea of what the rules of the house are.

What can be done now with grids

  • The grid system has always been available
  • Can be safely used for each little new feature
  • It's easy to refactor our code with mixins

We "correct" our layouts with semantic class names and we play with mixins

When a feature requires a layout change or when we need to build a new section, layout needs to be understood by both designers and developers

Mobile first approach

A site designed with RWD adapts the layout to the viewing environment by using fluid, proportion-based grids, flexible images and CSS3 media queries.
