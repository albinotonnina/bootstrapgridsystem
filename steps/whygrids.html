Why grid frameworks are important

Guess what, the scaffolding is where we need to work.

The vast majority of websites and applications use a grid. There might be not an explicit grid system in place but a simple two-column layout with a sidebar and a main content area is already a grid.

What those websites and application do is actually re-invent a new grid system around the product. This works.

But, when the layout has to evolve and we need to add complexity, it's not easy to maintain the initial stability. The grid becomes unreliable. We need to modify it on the fly.

Why grid frameworks are important

Grid frameworks such as Bootstrap's grid system are there to manage more complex layouts.

They also help when we need to work with complex environments, such SPA.

A grid system should be a structure at the lower levels of the layout in the optics of a modular scalable architecture for our HTML and CSS (SMACSS)

It means that should never be altered when composing and evolving the layout.

The grid is the mannequin and the comp is the pattern.

Working with grids allows designers and developers to easily communicate and understand how to put layouts together. It provides developers with units of measurement that they can easily turn into divs.

It also informs both sides how to layout elements on a page so that they can transition between different devices, mobile and desktop etc.

Benefits of grid frameworks

  • Visual harmony

  • A balanced and consistently implemented design scheme will increase readers’ confidence

  • Provides coherence to complexity

  • Easy maintenance

  • Communication developers/designers, smoother workflow

  • Responsive design

The goal is to establish a consistent, logical screen layout, one that allows us to “plug in” text and graphics without having to stop and rethink our basic design approach on each new page.

coherence to complexity: it's a matter of counting until 12.

easy maintenance: it's outside our code but always available.

communication: we sometimes communicate to each other with two slightly different unit of measurements, that leads to misunderstandings or pitfalls.
